HTML

What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language used for creating web pages which include text, graphics, and other support media. HTML uses various tags to structure the content and define the elements on a webpage. It is a simple markup language that describes the structure, layout, and behaviour of a web document. Html enables you to markup text so that it can function as hypertext on the web. this makes the text more interactive and dynamic.

History of HTML

HTML started its journey in the early 1930s with GML(General Markup Language). In 1986, this language was modified and was named SGML(Standard General Markup Language). In 1989, Tim Berners-Lee and his team redesigned this language and named it as HTML.

HTML Tags

HTML tags are used to mark up elements within an HTML document. Tags are enclosed in angle brackets (<>) and usually come in pairs, with an opening tag and a closing tag. The content between the opening and closing tags represents the element.

Example:

        
<p>This is a paragraph.</p>

Basic Structure of an HTML Document

        
<!DOCTYPE html>
        
<html>
        
<head>
            
<title>Page Title</title>
        
</head>
        
<body>
            
<h1>Heading 1</h1>
            
<p>This is a paragraph.</p>
       
</body>
        
</html>

Common HTML Elements

  • <h1> to <h6>: Headings
  • <p>: Paragraph
  • <a>: Anchor (link)
  • <img>: Image
  • <ul> and <li>: Unordered list
  • <ol> and <li>: Ordered list
  • <table>, <tr>, <th>, and <td>: Table and its components
  • <div>: Division or section
  • <span>: Inline element
  • <form> and <input>: Form and input fields

Elements

An HTML element represents one complete individual component. It consists of an ON tag, the content, and an OFF tag.
The HTML elements can be categorised as:

  • Container Elements:- The elements that include both ON tag and OFF tag are called Container Elements. The OFF tag has the same name as that of an ON tag. The difference between the ON tag and the OFF tag is that the OFF tag starts with a '/' symbol.
  • Empty Elements:- These elements only contains ON tags. They do not have OFF tags. These elements do not enclose any data ; instead, they do some function on their own.

Example:

        
<B>This makes the text bold.</B>- Container Tag
        
It is used as s separator between the blocks of text.<Hr>- Empty Tag

Attributes

HTML tags can have attributes that provide additional information about the element. Attributes are specified within the opening tag and have a name and a value.

Example:

        
<a href="https://www.example.com">Link</a>
        
<img src="image.jpg" alt="Image">

CSS and JavaScript

CSS (Cascading Style Sheets) is used to style HTML elements and control their appearance, while JavaScript is used to add interactivity and dynamic behavior to web pages.

Conclusion

HTML is the foundation of web development. Understanding the basics of HTML tags, structure, and common elements is essential for creating well-structured web pages.
