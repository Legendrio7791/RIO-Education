CSS

Introduction

CSS stands for Cascading Style Sheets. It is a styling language used to describe the look and formatting of a document written in HTML.

CSS allows you to separate the presentation of your web pages from the structure and content.

Basic CSS Syntax

CSS rules consist of a selector and a declaration block. The selector specifies which HTML elements to style, and the declaration block contains the property-value pairs that define the styles.

Example:


    selector {

                        property: value;


         }

Selectors

Selectors determine which elements to target with CSS styles. There are various types of selectors:


  • Element selector: targets elements by their HTML tag name, e.g., p selects all paragraphs.
  • Class selector: targets elements with a specific class attribute, e.g., .highlight selects all elements with class "highlight".
  • ID selector: targets an element with a specific ID attribute, e.g., #intro selects the element with ID "intro".
  • ...and more (e.g., attribute selectors, pseudo-classes, etc.)

Properties and Values

Properties specify the aspect of an element to be styled, and values define how the property should be applied.

Example:

color: navy; sets the text color to navy,
background-color: yellow; sets the background color to yellow.

Cascading and Specificity

CSS follows the "cascading" principle, where multiple styles can be applied to the same element, and the final result is determined by a set of rules known as specificity.

If multiple rules conflict, the most specific rule takes precedence. Inline styles have the highest specificity, followed by IDs, classes, and element selectors.

Conclusion

CSS is a powerful tool for controlling the visual appearance of web pages. By using CSS, you can make your HTML documents more engaging and user-friendly.

Remember to experiment and explore the various CSS properties and selectors to create unique and compelling designs!
